# AGENTS.md

## Project Goal

OpenVLA LoRA fine-tuning and LIBERO evaluation demo for robotics research portfolio.

## Code Principles

- Keep changes minimal.
- Prefer official OpenVLA commands.
- Do not invent experimental results.
- Do not claim training has been completed unless logs exist.
- Do not download large models or datasets unless explicitly requested.
- Do not run training unless explicitly requested.
- Put environment reports under `notes/environment_report.md`.
- Put experiment logs under `notes/experiment_log.md`.
- Put resume-ready bullets under `notes/resume_bullets.md`.

## Default Constraints

- Use Python 3.10 where possible.
- Prefer LoRA instead of full fine-tuning.
- Prefer LIBERO before BridgeData V2.
- Use small debug runs first.
